Prototype

Forces


Multiple instances of the same object needed in an optimal way.


Solution


Get a clone of the object you want to make multiple instances of and keep a clone
of said object instead of recreating the object from abstract factory / concrete factory.


Related Patters


  • Factory
  • Abstract Factory


Example (Pseudo Code)

//returns a Red,Blue, or green Square,Circle,Triangle.
Shape shapeFactory(string shape,string color);

//A Dictionary to hold shapes, key : value
Dictionary dict = new Dictionary();
dict.put("square1":shapeFactory("square","red"));

Shape sq1 = dict.get("square1");
Shape sq2 = dict.get("square1");
Shape sq3 = dict.get("square1");
