Builder

Forces


Need to build a complex object in an easy way.


Solution


Build the object step by step with smaller objects.


Related Patters


  • Factory Method
  • Composite
  • Prototype
  • Singleton


    • Example (Pseudo)


    //We want to build a car several objects.
    //We just tell the Car class the name of the car and the builder does the rest.
    class CarBuilder(){
         Engine engine;
         Frame frame;
         Transmission trans;
         string name;

         void CarBuilder(string carName){
               name=carName;
               engine = new Engine(carName); //holds all engine specs
               fram = new Frame(carName); //holds all Frame specs
               trans = new Transmission(carName); //holds all Transmission specs
         }
    }
