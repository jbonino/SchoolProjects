Factory Method

Forces


A need to create multiple instances of a new object.


Solution


We create a concrete factory object which we can call to have an object constructed and then passed to us.
The factory object hides creation logic.


Related Patters


  • Abstract Factory
  • Builder

Example (Pseudo Code)

Shape shapeFactory(string shape,string color){
      if(shape=="square")
           return new square(color);
      if(shape=="triangle")
           return new triangle(color);
      if(shape=="triangle")
           return new triangle(color);
      return NULL;
}
