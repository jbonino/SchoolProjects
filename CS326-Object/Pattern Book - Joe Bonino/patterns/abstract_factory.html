Abstract Factory

Forces


A need to create an instance(s) of a new object. Client needs appropriate object for system context.


Solution


Have a abstract factory object holding concrete factories. Abstract factory knows the appropriate
object to return.


Related Patters


  • Factory Method
  • Builder
  • Prototype


Example (Pseudo Code)


//Abstract factory holds a factory for windows and mac
Define Abstract_GUI_Factory(Specs s){
     if(OS == windows ) return GUI_WindowsFactory(Specs s);
     if(OS == mac) return GUI_MacFactory(Specs s);
}
