An existing object needs a new function(s) but we dont want to change its original structure.


Solution


Create a new object to "wrap" the already existing object. This new object will keep existing
object's methods' intact but will add additional functionality.

  • Adapter


Example (Pseudo)


//We have a circleFactory(int radius) that builds a circle and returns it.
//We want to add a color to the circle without manipulating the circleFactory method

class colorCircleFactory(){

     ColorCircle create(string color,int radius){
           return new ColorCircle("red",circleFactory(radius)); //ColorCircle holds a color and Circle object
     }
}

class ColorCircle(){
     private Circle circle;
     private string color;

     void ColorCircle(string color,Circle circle){
           this.circle= circle;
           this.color = color;
     }
