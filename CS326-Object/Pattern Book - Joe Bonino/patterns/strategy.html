Strategy

Forces


A need to use one of several behaviors dynamically.


Solution


Create objects which represent various strategies and a context object whose behavior varies as per its strategy object.


Related Patters


  • Fly Weight
  • Template Method


Example (Pseudo Code)

Class Human{
     string name;
     int age;
     Job j;

     string askJob(){ return j.job(); }
     void setJob(Job newJ){ j=newJ; }

}

interface Job{
     string job();
}

class hasJob implements Job{
     string job(){ return "I have a job!"; }
}

class noJob implements Job{
     string job(){ return "I dont have a job!"; }
}

class Joe extends Human{
     void Joe(){
          Job = new noJob(); //can dynamically change later with setJob method.
     }
}
