Composite

Forces


A bunch of similar functions are needed to complete a task.


Solution


Put these functions in their own objects responsibly and cordinate them in a tree structure.


Related Patters


  • Visitor
  • Chain of Responsibility
  • Decorator


Example (Pseudo)

//Our employee class composes of a name,dept,salary, and subordinates
class Employee {
     string name;
     string dept;
      int salary;
      List<Employee> subordinates;

     public Employee(String name,String dept, int sal) {
           this.name = name;
           this.dept = dept;
           this.salary = sal;
           subordinates = new ArrayList<Employee>();
     }

     public void add(Employee e) {
           subordinates.add(e);
     }

     public void remove(Employee e) {
           subordinates.remove(e);
     }

     public List<Employee> getSubordinates(){
           return subordinates;
     }
}
