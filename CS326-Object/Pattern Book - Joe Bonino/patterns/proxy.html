Proxy

Forces


Want to reduce the memory footprint of loading an object. Only initialize when object needed.


Solution


Make a object that represents the functionality of another object. Let object 'A' be our new Proxy object
and object 'B' is the represented object. 'A' will hold 'B's object that lets it communicate to outside world.


Related Patters


  • Adapter
  • Decorator
  • Lazy Initialization


Example (Pseudo)


//We create a JPEG object that holds an image and can display itself.
//Now we us a proxyJPEG to preload the image and JPEG will be instantiated.

class proxyJPEG(){

     JPEG image;
     string filepath;

     constructor(f){
           filepath=f;
     }

     void display(){
           if(!image) image = new JPEG(filepath);
           image.display();
     }

}
