Wrapper

Forces


A object needs more functionality but cant tamper with objects structure.


Solution


Create a wrapper object that holds the same functionality of original object but then also adds some.


Related Patters


  • Adapter
  • Bridge


Example (Pseudo Code)

public class Parser{
     String rawInput
     public Parser(String s){ rawInput=s; }

     public String parseString(){
          String parsedString = //do some parsing on rawInput
          return parsedString;
     }
}
//we need the rawInput from parser for some reason

public class ParserWrapper{ //wrapper class
     String rawInput;
     Parser parser;
     public ParserWrapper(String s){
          rawInput=s;
          parser = new Parser(s);
     }
     public String parseString(){ return parser.parseString; }
     public String rawString(){ return rawInput; }
}
