Fly Weight

Forces


A need to reduce the number of similar objects created for better performance.


Solution


Reuse already existing similar kinds of objects by storing them and create new a object when no matching object is found.


Related Patters


  • Facade
  • Composite


Example (Pseudo)


class SquareFactory(string color){
     HashTable table = new HashTable(); //key is color, table holds Squares

     createSquare(string color){
           Square s = table.get(color);
           if(!s){
                s = new Square(color);
                table.put(s);
           }
      return s;
     }
}
