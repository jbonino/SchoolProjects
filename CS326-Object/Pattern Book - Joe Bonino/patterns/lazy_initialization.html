Lazy Initialization

Forces


An object isnt needed to exist until a method is called.


Solution


Create the object when method is called.


Related Patters


  • Proxy


Example (Pseudo)


//A program was created to show what user ate everyday and provide nutrition facts.
//We have an object Apple that holds all information about it (calories, sugars, etc).

Apple a;
string ateApple(int quantity){
     if(!a) Apple a = new Apple();
     string nutrion_facts = a.nutrition_facts(quantity);
     return nutrition_facts;
}
