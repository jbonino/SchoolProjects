Null Object

Forces


Program crashes when something is Null and used in program.


Solution


Create a Null object that does nothing instead of crashing program. This object extends an abstract class and is used in replace of Null.


Related Patters


  • Singleton
  • State
  • Visitor


Example (Pseudo Code)

public abstract class AbstractWeapon {
     private String name;
     public abstract boolean isNull();
     public abstract String getName();
}

public class Axe extends AbstractWeapon{

     public Axe(string name){ this.name=name; }
     @Override
     public String getName(){return name;}
     @Override
     public isNull(){ return false };
}

public class NullWeapon extends AbstractWeapon{
     @Override
     public String getName(){"Weapon does not exist.";}
     @Override
     public isNull(){ return true };
}
